# 期末刷題網站

這是一個純前端刷題網站，可部署到 GitHub Pages。

## 內容

- 第 7 章 14 題
- 卷 05-08 題庫 175 題
- 總題數：189 題
- 支援圖片題
- 支援全部題目、隨機練習、錯題練習
- 錯題會存在手機/瀏覽器的 localStorage

## 部署

1. 把本資料夾內所有檔案上傳到 GitHub Repo 根目錄。
2. 進入 Settings → Pages。
3. Source 選 Deploy from a branch。
4. Branch 選 main，Folder 選 /root。
5. 儲存後等待部署完成。

## 注意

`images/` 資料夾一定要一起上傳，否則圖片題會破圖。
