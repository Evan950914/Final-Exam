# 第 7 章期末刷題網站

這是一個純前端刷題網站，可部署到 GitHub Pages。

## 檔案

- `index.html`：網頁入口
- `style.css`：手機版樣式
- `script.js`：刷題邏輯
- `questions.js`：題庫
- `images/`：題目圖片

## 功能

- 全部題目練習
- 隨機練習
- 錯題本
- 顯示答案串
- 手機可用
- 不依賴後端、不需要資料庫

## 部署到 GitHub Pages

1. 把這個資料夾裡的所有檔案上傳到 GitHub Repo 根目錄。
2. 到 `Settings` → `Pages`。
3. Source 選 `Deploy from a branch`。
4. Branch 選 `main`，Folder 選 `/root`。
5. 等 1 到 3 分鐘。

網站網址通常會是：

```txt
https://你的帳號.github.io/你的Repo名稱/
```
